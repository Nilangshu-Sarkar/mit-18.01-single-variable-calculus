# **MIT 18.01SC: Selected Problem Set Solutions**

### *Conceptual Calculus for Physics | Independent Study by Nilangshu Sarkar*

This repository contains a curated set of solutions from **MIT 18.01SC Single Variable Calculus**, completed as part of my self-directed preparation for theoretical physics.

I did **not** attempt to solve every problem in the course.
Instead, I **selectively solved** the problems that strengthened:

* conceptual understanding beyond high-school calculus
* intuition for physical applications (motion, geometry, arc length, volumes)
* problem-solving depth required for 8.01/8.02/8.03 and advanced physics study
* LaTeX writing and mathematical communication skills

The goal here is *not* to showcase exhaustive coverage, but to document a **focused learning path** that bridges high-school calculus to the level needed in physics research.
